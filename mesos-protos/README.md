# Protocol buffer serializer and deserializer for Apache Mesos

[![Build Status](https://travis-ci.org/PROTEINE-INSAIDERS/mesos-client.svg?branch=master)](https://travis-ci.org/PROTEINE-INSAIDERS/mesos-client)
